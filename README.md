# Energy Consumption Analysis Dashboard

## Overview

This project presents an interactive Energy Consumption Analysis Dashboard built using AWS, Snowflake, and Tableau. The dashboard analyzes energy consumption patterns and provides insights into usage trends, consumption distribution, and key performance indicators.

The project demonstrates a modern cloud-based analytics workflow, where data is stored and processed in Snowflake, integrated through AWS infrastructure, and visualized using Tableau.

## Tools & Technologies

* AWS
* Snowflake
* Tableau

## Key Features

* Energy Consumption Analysis
* Usage Trend Monitoring
* Consumption Distribution Insights
* Interactive Filters and Dashboards
* KPI Tracking and Reporting
* Data-Driven Decision Support

## Project Workflow

1. Load and store energy consumption data.
2. Manage and query data using Snowflake.
3. Utilize AWS cloud services for data infrastructure.
4. Connect Snowflake to Tableau.
5. Create interactive dashboards and visualizations.


## Objective

The objective of this project is to analyze energy consumption data and transform raw data into actionable insights through an interactive Tableau dashboard powered by AWS and Snowflake.
